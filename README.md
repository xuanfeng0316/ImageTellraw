# ImageTellraw

Convert images to Minecraft Java Edition tellraw commands for in-game visualization.

---

## Note

In vanilla Minecraft, 100% restoration is not possible.  
The in-game display effect cannot reach high precision.  
It can only guarantee "recognizable", not "detailed".

---

## Usage

The main program is `main.py`.

Run:

```bash
python main.py
```

The output command will be written to `CommandOutput.txt`.

Please execute the command in a **command block** in-game.  
The chat bar may not be able to accommodate the full command.

---

## Example Images

The `Example_Input_Picture/` directory contains sample images.  
If you just want to test it immediately, you don't need to prepare your own image.

For copyright and usage details, please refer to `Example_Input_Picture/Notice.txt`.
